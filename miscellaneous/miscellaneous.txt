Easy Questions:

1. Write a Python program to print "Hello, World!" to the console.
2. Calculate the sum of all even numbers from 1 to 100.
3. Write a Python function that checks if a given number is prime.
4. Calculate the factorial of a number using a loop.
5. Write a Python program to reverse a given string.

Medium Questions:

6. Write a Python program to find the Fibonacci sequence up to the nth term.
7. Implement a function to check if a given string is a palindrome.
8. Create a Python class representing a basic calculator with add, subtract, multiply, and divide methods.
9. Write a program to find the intersection of two lists in Python.
10. Implement a simple text-based game where the player guesses a random number generated by the computer.

Difficult Questions:

11. Write a Python program to solve the Tower of Hanoi problem using recursion.
12. Create a Python program that simulates a basic banking system with user accounts and transactions.
